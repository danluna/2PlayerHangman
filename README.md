Two Player Hangman

Connect to the game with a partner and collaboratively attempt to guess the word before your tries run out. 

http://twoplayerhangman.herokuapp.com/

Run:
  - node app.js
  - Connect on localhost:3000

Tech: 
  - Socket.io 
  - Node.js 
  - Express.js
  - jQuery
  - Bootstrap
  - ejs
